Commands

  • ATOM_TYPE is important and must correspond to the numbers on sixth line of the POSCAR
  • PRIMITIVE_AXIS needs to be used to map supercells built from conventional cells onto primitive cell band structures
  • Create the displaced structures for a defective cell: phonopy -d --dim='1 1 1' --tolerance=1e-4
  • If using the tolerance tag this must continue to be used for the FORCE_SETS generation and any subsequent band/dos calculations
  • Create FORCE_SETS: phonopy -f disp/{001...008}/vasprun.xml --tolerance=1e-3
  • the disp.yaml will need to be in current folder
  • plot band: phonopy -s -p band.conf --tolerance=1e-3
  • the POSCAR will have to be in the current folder
  • plot dos/thermal: phonopy -s -p (-t) dos.conf

Thanks to Adam, it’s super easy to generate lovely .gif files of the phonon modes

2) Call ascii-phonons to do it’s magic:
